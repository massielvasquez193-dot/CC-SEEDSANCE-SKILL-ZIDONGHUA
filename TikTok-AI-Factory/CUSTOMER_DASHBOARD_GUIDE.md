启动软件后，点击顶部标签栏的「👤 客户中心」即可进入。

## 各模块说明

### 授权信息
- 查看当前 License Key 和到期时间
- 到期前 30 天显示黄色警告
- 已过期显示红色警告
- 点击「续费」查看续费联系方式

### 使用统计
- 自动扫描 output 目录统计视频数量
- 按今日/本周/本月/累计维度展示

### 成本统计
- 基于视频数量和单价估算
- 细分到每个 AI 服务的成本
- 实际成本以 API 账单为准

### 视频记录
- 最近 100 条生成记录
- 点击 📂 打开对应输出目录
- 点击 ▶ 直接播放视频
- 可导出 CSV 进行离线分析

### 系统状态
- ✓ 绿色 = 正常
- ✗ 红色 = 异常
- — 灰色 = 未配置

## 常见问题

**Q: 成本数据准确吗？**
A: 成本基于视频数量和标准单价估算，实际费用以各 API 服务商账单为准。

**Q: 如何导出数据？**
A: 点击底部的「📊 导出 CSV」按钮，选择保存位置即可。

**Q: 数据多久更新一次？**
A: 切换到客户中心标签页时自动刷新，也可点击「🔄 刷新」手动更新。
